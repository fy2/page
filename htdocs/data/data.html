Genome Viewer

Use Artemis to browse or annotate genomes

Artemis is a genome browser and annotation tool that allows visualisation of sequence features within the context of the sequence, and also its six-frame translation.

Genome Comparison

Use Artemis Comparison Tool (ACT)

ACT is a tool for displaying pairwise comparisons between two or more DNA sequences.

Phylogeography

Visualise differences within the context of geographical location

Spatial Viewer was developed by the London Imperial College. It displays variations between genomes in a geographical context along with the relevant SNP information.
